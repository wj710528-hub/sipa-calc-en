Average screen illuminance guide (engineering reference)

120–200 luxCinema / dark room
200–400 luxHome theater / dim room
400–650 luxMeeting or classroom / controlled ambient
650–1000 luxBright indoor / exhibit
>1000 luxSpecialty projection assessment
This version is optimized for quick estimates. The primary metric is theoretical average screen illuminance (lux). nit and fL remain secondary reference values. All results are theoretical and are still affected by ambient light, screen properties, lens transmission, correction loss, uniformity and system contrast. The lux value here means screen white-field illuminance, not room ambient illumination.
